Новый формат сайта¶

Изменен формат сайта. Блог будет разрабатываться и обновляться на основе ABlog для разработки блогов на Sphinx.

Все материалы созданы с помощью системы Sphinx. PDF файл генерируется с использованием LaTeX, а в HTML для отображения формул используется MathJax. В связи с этим возможны некоторые проблемы при отображении формул, а также возможны ошибки в тексте, поэтому сообщайте мне об этом, пожалуйста.
