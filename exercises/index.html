Зачет заданий¶

Для выполнения упражнений студент должен зарегистрироваться на GitHub, создать репозитарий, в котором каждому разделу должна соответствовать директория, внутри которой располагаются упражнения и задачи (каждая в своей директории) из соответствующего раздела.

Предоставить мне доступ к репозитариию, чтобы я мог проверять выполнение упражнений.

Хороший вариант предоставления доступа¶

Добавить меня в Collaborators репозитария (Settings > Collaborators) мое имя на GitHub — slemeshevsky

Другой вариант предоставления доступа¶

Отправить мне ссылку на ваш репозитарий по email: sergey.lemeshevsky@gmail.com

Примерная структура репозитария¶

repo
|- ch1
    |- ex-1
        |- ex-2
        |- ex-3
             .
             .
        |- pr-1
       .
       .

О сроках выполнения упражнений будет сообщаться на лекциях и на этом сайте. Каждое из упражнений, выполненное в срок, будет засчитываться с баллом 1. Те упражнения, которые будут выполнены позже срока, засчитываются с баллом 0.5.

Таким образом получится что-то вроде рейтинга, который будет учтен при зачете и экзамене (пока точно не знаю будет у нас два экзамена, либо один экзамен и зачет).
