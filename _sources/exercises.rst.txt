Зачет заданий
=============

Для выполнения упражнений студент должен зарегистрироваться на `GitHub
<https://github.com>`_, создать репозитарий, в котором каждому разделу
должна соответствовать директория, внутри которой располагаются
упражнения и задачи (каждая в своей директории) из соответствующего
раздела.

Предоставить мне доступ к репозитариию, чтобы я мог проверять
выполнение упражнений.

Хороший вариант предоставления доступа
--------------------------------------

Добавить меня в Collaborators репозитария (:menuselection:`Settings --> Collaborators`)
мое имя на GitHub --- slemeshevsky

Другой вариант предоставления доступа
-------------------------------------

Отправить мне ссылку на ваш репозитарий по email: ``sergey [dot] lemeshevsky [at] gmail.com``


Примерная структура репозитария
-------------------------------
   
:fa:`minus-square-o` :fa:`folder` ch1
	
	:fa:`file-code-o` - filename1.py

	:fa:`file-code-o` - filename2.py

	:fa:`file-code-o` - filename3.py
		
	...

:fa:`minus-square-o` :fa:`folder` ch2
	
	:fa:`file-code-o` - filename1.py

	:fa:`file-code-o` - filename2.py

	:fa:`file-code-o` - filename3.py
		
	...

...
	
О сроках выполнения упражнений будет сообщаться на лекциях и на этом
сайте.

У нас будет в каждом семестре зачет и экзамен. Для сдачи зачета (и,
соответственно, допуска к экзамену) нужно выполнить все упражнения. На
основе выполненных упражнений будет выставлен зачет. Если в каком-то
из-заданий меня что-то будет не устраивать или возникнет вопрос, то к
репозитарию будет добавлен открытый issue (задача) в репозитарии. До
того момента пока этот issue не будет закрыт, задание не будет
считаться сданным.
